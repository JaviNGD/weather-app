## Run Locally 

Clone the project
```plaintext
git clone https://github.com/JaviNGD/weather-app.git
```

Go to the project directory
```plaintext
cd weather-app
```

Install dependencies
```plaintext
  npm install
```

Start the server
```plaintext
  npm start
```
